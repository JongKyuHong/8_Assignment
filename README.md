# 8번 과제

- 기존 온라인 강의에서 확장한 버전
- 동전을 먹으면 레벨업하는 로직을 추가하였다.
- 레벨업을 하면 캐릭터의 능력을 올려주는 증강을 고를 수 있음
- 아이템은 큰 동전, 작은 동전, 힐 포션, 지뢰, 독 포션, 슬로우 포션이 있음

- 동전을 일정량 먹으면 레벨업
- 증강을 고를 수 있음
- 증강은 실버, 골드, 프리즘이라는 등급이 나뉨
  - 등급이 높을 수록 좋은 효과
